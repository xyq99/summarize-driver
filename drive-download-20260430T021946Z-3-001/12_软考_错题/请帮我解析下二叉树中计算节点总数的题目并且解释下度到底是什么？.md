第一题是一道非常经典的二叉树计算题，很多考生容易在这里“凭直觉”猜答案，但其实只要掌握了“度”的概念和两个核心公式，这类题就是送分题。

### 1. 什么是“度” (Degree)？

在树形结构中，“度”是一个描述节点“生了几个孩子”的量化指标。

- **节点的度：** 指该节点拥有的**子树数目**（即直接相连的下级节点个数）。

  - **度为 0 的节点：** 没有孩子，我们称之为**叶子节点**（Leaf）。

  - **度为 1 的节点：** 只有一个孩子。

  - **度为 2 的节点：** 有左、右两个孩子。

- **树的度：** 指树中所有节点中，**度的最大值**。对于二叉树来说，树的度最大只能是 2。

### 2. 第一题深度解析

**题目：一棵完全二叉树共有 30 个节点，求其叶子节点（度为 0）数。**

我们要用到二叉树的两个基本公式：

1.  **节点总数公式：** \$n = n_0 + n_1 + n_2\$  
    *(总节点 = 度为0的 + 度为1的 + 度为2的)*

2.  **叶子节点与双亲节点关系：** \$n_0 = n_2 + 1\$  
    *(这是一个恒等式：叶子节点总是比度为 2 的节点多一个)*

#### **推导过程：**

将公式 (2) 代入公式 (1)：

\$\$n = (n_2 + 1) + n_1 + n_2\$\$

\$\$n = 2n_2 + 1 + n_1\$\$

已知总数 \$n = 30\$，代入得：

\$\$30 = 2n_2 + 1 + n_1 \Rightarrow 29 = 2n_2 + n_1\$\$

#### **关键点：完全二叉树的特殊性**

在**完全二叉树**中，度为 1 的节点（\$n_1\$）只有两种可能：**0 个或 1 个**。

- 如果 \$n_1 = 0\$：\$29 = 2n_2 + 0 \Rightarrow n_2 = 14.5\$（节点数必须是整数，不成立）。

- 如果 \$n_1 = 1\$：\$29 = 2n_2 + 1 \Rightarrow 2n_2 = 28 \Rightarrow n_2 = 14\$。

现在我们求出了 \$n_2 = 14\$，再根据公式 \$n_0 = n_2 + 1\$：

\$\$n_0 = 14 + 1 = 15\$\$

所以，**叶子节点数是 15**。

### 3. 快速得分技巧（秒杀法）

在软考中，如果你看到**总节点数 \$n\$ 是偶数**（如本题的 30）：

- 该完全二叉树一定有一个度为 1 的节点。

- **叶子节点数 \$n_0 = n / 2\$**。即 \$30 / 2 = 15\$。

如果你看到**总节点数 \$n\$ 是奇数**（如 31）：

- 该完全二叉树度为 1 的节点为 0。

- **叶子节点数 \$n_0 = (n + 1) / 2\$**。即 \$(31 + 1) / 2 = 16\$。
