## 模拟题考点与答案深度解析

### 1. 计算机系统基础

- **题目概要**：流水线由于相关冲突会导致吞吐率下降。多操作同时争用同一资源（如同一内存单元）引起的冲突属于哪种冲突？

- **正确答案**：**结构相关（Structural Hazard）**

- **核心分析**：流水线冲突主要分为三类：**结构相关**（硬件资源争用，如单引脚内存同时读指令和读数据）、**数据相关**（RAW/WAR/WAW 等数据依赖，前条指令没写回，后条指令就要读）和**控制相关**（由跳转指令、分支预测失败引起）。

### 2. 嵌入式系统软件

- **题目概要**：关于板级支持包（BSP）的叙述中，哪项是正确的？

- **正确答案**：**BSP 介于硬件和操作系统之间，其主要功能是实现硬件初始化和提供底层驱动**

- **核心分析**：BSP 属于系统软件，不是应用软件。它向上为操作系统内核提供统一的驱动接口，向下屏蔽底层硬件细节。即使硬件相同，针对 Linux 和 RT-Thread 的 BSP 代码和接口规范也是完全不同的。

### 3. 操作系统知识

- **题目概要**：在实时操作系统（RTOS）中，为了防止高优先级任务被低优先级任务长期阻塞（优先级翻转），通常采用什么机制？

- **正确答案**：**优先级继承协议（PIP）**

- **核心分析**：**优先级翻转**是 RTOS 的经典漏洞。当高优先级任务 A 等待被低优先级任务 C 占有的互斥锁时，中等优先级任务 B 抢占了 C，导致 A 间接被 B 长期阻塞。解决办法是让 C 临时**继承** A 的高优先级，使其赶快执行完并释放锁。

### 4. 网络基础知识

- **题目概要**：在 TCP/IP 协议栈中，ARP（地址解析协议）工作时主要通过什么方式发送请求？

- **正确答案**：**广播**

- **核心分析**：ARP 的核心任务是将 IP 地址（网络层）转换为 MAC 地址（数据链路层）。因为在发送请求时，源主机并不知道目标主机的 MAC 地址，所以必须在局域网内通过**广播（FF:FF:FF:FF:FF:FF）寻问，而目标主机得知是找自己后，会用单播**进行回应。

### 5. 安全性基础知识

- **题目概要**：截获合法用户的合法报文后重新发送，从而欺骗接收系统的攻击被称为？

- **正确答案**：**重放攻击（Replay Attack）**

- **核心分析**：重放攻击属于被动攻击中的录制与二次发送，哪怕数据被加密，攻击者不需要解密，直接原样重发也能达到欺骗效果。防御此攻击通常在报文中引入**时间戳（Timestamp）或一次性随机数（Nonce）**。

### 6. 嵌入式软件程序设计

- **题目概要**：C 语言嵌入式编程中，哪种场景下**不需要**显式将变量声明为 volatile？

- **正确答案**：**多线程或多任务并发环境中，由多个任务共享且互斥访问的普通局部变量**

- **核心分析**：volatile 告诉编译器该变量可能在外部被改变，禁止将其优化到寄存器中。**硬件寄存器**、**中断（ISR）与主循环共享的全局变量**、**DMA 缓冲区的指针**都必须加 volatile。而**局部变量**分配在栈上或局部寄存器里，生命周期仅在函数内部，不能被多任务直接共享。

### 7. 嵌入式系统实施知识

- **题目概要**：验证软件单元是否符合详细设计规格，并在白盒测试环境下观察代码覆盖率的测试阶段是？

- **正确答案**：**单元测试（Unit Testing）**

- **核心分析**：软件测试级别由低到高为：**单元测试**（对照详细设计，白盒测试为主，看覆盖率）、**集成测试**（对照概要设计，测模块接口）、**系统测试**（对照需求规格，黑盒测试整体功能/性能）和**验收测试**（用户主导）。

### 8. 数据结构和算法

- **题目概要**：资源受限系统中，实现先进先出（FIFO）队列，如何避免频繁数据搬移并高效利用连续内存？

- **正确答案**：**循环队列（基于数组实现）**

- **核心分析**：普通顺序表在出队时需要把后续元素全部向前平移，时间复杂度高达 \$O(n)\$。链表虽然不需要搬移，但每个节点自带指针域，在嵌入式中会带来额外的内存开销和频繁申请释放产生的内存碎片。**循环队列**利用固定大小数组配合模运算（%）让头尾相接，效率最高。

### 9. 数字电路基础

- **题目概要**：下列哪一个器件属于组合逻辑电路？

- **正确答案**：**全加器（Full Adder）**

- **核心分析**：**组合逻辑电路**（如全加器、编码器、译码器、数据选择器）的输出仅由当前的输入决定，内部没有记忆元件。而**时序逻辑电路**（如 D 触发器、寄存器、计数器）内部包含存储元件，输出不仅取决于当前输入，还取决于过去的状态。

### 10. 嵌入式系统的存储体系

- **题目概要**：初次访问某个数据块时必然会发生的 Cache 未命中，被称为？

- **正确答案**：**冷启动不命中 / 强制不命中（Compulsory Miss）**

- **核心分析**：Cache 的“3C”失效模型包括：**强制不命中**（冷启动首次访问，无可避免）、**容量不命中**（Cache 总容量太小，装不下程序工作集）和**冲突不命中**（由于组相联或直接映射限制，多个主存块争用同一个 Cache 槽位导致被交替替换）。

### 11. 嵌入式系统总线和通信接口

- **题目概要**：SPI 通信中，控制数据在时钟哪个边缘采样以及时钟空闲时电平状态的两个关键配置参数是？

- **正确答案**：**时钟极性（CPOL）与时钟相位（CPHA）**

- **核心分析**：**CPOL**（Clock Polarity）决定时钟空闲时是高电平还是低电平；**CPHA**（Clock Phase）决定是在时钟的第一个跳变沿还是第二个跳变沿采样。这两个参数的不同组合构成了 SPI 的 4 种工作模式。而设备地址和 ACK 是 I2C 总线的特征。

### 12. 嵌入式硬件设计

- **题目概要**：要求处理器在挂起时能保留内部 RAM 的数据，且能通过外部中断快速唤醒，应选择哪种电源模式？

- **正确答案**：**停止 / 挂起模式（Stop / Suspend Mode）**

- **核心分析**：现代处理器有多种低功耗模式：**睡眠模式**仅关内核时钟，省电有限；**掉电/深度休眠模式**会彻底切断 RAM 供电以追求极致省电，但唤醒类似冷启动，数据丢失。**停止/挂起模式**关闭 CPU 和多数外设时钟，但保持 RAM 供电，支持外部中断瞬间唤醒，是兼顾数据完整与快速响应的最佳平衡。

### 13. 知识产权与标准化

- **题目概要**：下列标准代号中，属于我国强制性国家标准的是？

- **正确答案**：**GB**

- **核心分析**：在我国国家标准代号中：**GB** 代表国家强制性标准（必须严格执行）；**GB/T** 代表推荐性标准（Recommended，鼓励采用）；**GB/Z** 代表国家标准化指导性技术文件（Guidance）。

### 14. 多媒体知识

- **题目概要**：采样率 44.1 kHz、量化 16 位、双声道，录制 1 秒钟未压缩的原始 PCM 音频需要多少存储空间？

- **正确答案**：**176,400 字节**

- **核心分析**：计算公式为：  
  \$\$\text{数据量(Byte)} = \text{采样频率(Hz)} \times \left(\frac{\text{量化位数(bit)}}{8}\right) \times \text{声道数} \times \text{时间(s)}\$\$  
  代入数据：\$44,100 \times \left(\frac{16}{8}\right) \times 2 \times 1 = 176,400 \text{ 字节}\$。如果在选择中选了 1,411,200，那是比特率（bps），需要除以 8 换算成字节。

### 15. 专业英语

- **题目概要**：看懂 Watchdog Timer (WDT) 完形填空：“... if the main program fails to \_\_\_\_\_\_\_ the counter periodically due to a software crash ...”

- **正确答案**：**clear / refresh**

- **核心分析**：这句话描述的是看门狗定时器的工作原理：如果主程序因为软件崩溃或死循环，导致无法定期**清除/刷新**（俗称“喂狗”）计数器，看门狗电路就会引发系统复位（Reset）。
