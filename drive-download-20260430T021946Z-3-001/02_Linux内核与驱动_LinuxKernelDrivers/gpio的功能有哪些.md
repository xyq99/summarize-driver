GPIO（General-Purpose Input/Output，通用型输入/输出）是芯片最基础、最灵活的引脚功能。对于 RK3568 这种复杂的 SoC，一个 GPIO 引脚往往身兼数职。

我们可以将 GPIO 的功能分为**基础数字功能**、**电气配置功能**和**复用功能**三大类。

## 1. 基础数字功能 (Basic I/O)

这是 GPIO 最本质的用法，通常只有两种逻辑状态：0（低电平）或 1（高电平）。

- **输入模式 (Input)：**

  - **电平检测：** 读取引脚上的高低电平。常用于检测按键按下、传感器开关状态（如红外对管）或外部芯片的状态信号（如 Ready 信号）。

  - **中断触发 (Interrupt)：** 这是输入的高级用法。当引脚电平发生变化（上升沿、下降沿或高低电平）时，通知 CPU 暂停当前任务转而处理紧急事件。

- **输出模式 (Output)：**

  - **电平控制：** 由 CPU 决定输出 0 还是 1。常用于控制 LED 灯的亮灭、蜂鸣器鸣叫、继电器闭合或给其他芯片提供使能信号（Enable）。

## 2. 电气配置功能 (Electrical Configuration)

为了适配不同的外部电路，GPIO 内部集成了多种电子元件进行调节。

- **上下拉配置：** 如前所述，配置内部上拉或下拉电阻，确保信号在悬空时有确定的逻辑状态。

- **驱动能力调整 (Drive Strength)：** \* RK3568 的 GPIO 通常可以设置不同的电流等级（如 2mA, 4mA, 8mA, 12mA）。

  - **作用：** 如果负载较重（如驱动长线缆或高速信号），需要调大驱动电流以保证波形不失真。

- **高阻态 (High-Impedance / Tri-state)：** \* 引脚既不输出高也不输出低，相当于断开连接。常用于总线结构，防止多个设备输出冲突。

## 3. 复用功能 (Multiplexing / Pinmux)

这是 SoC 芯片最强大的地方。因为芯片引脚有限，一个物理引脚可以通过内部选择器切换功能。

- **硬件控制器连接：** 一个 GPIO 引脚可以被配置为 UART 的 TX、I2C 的 SDA、PWM 的输出口或 SPI 的片选线。

  - *例如：* RK3568 的某个引脚，在功能 0 时是普通 GPIO，功能 1 时可能是 UART3_TX，功能 2 时可能是 PWM5。

- **模拟功能 (Analog)：** 部分特定的 GPIO 内部连接到了 ADC（模数转换器），可以用来读取连续变化的电压值（如滑动变阻器或电池电压），此时它不再是数字引脚。

## 4. 特殊高级功能

- **唤醒功能 (Wakeup)：** 在 SoC 进入深度睡眠（Deep Sleep）模式时，只有特定的 GPIO 还能保持检测能力。当外部信号触发这些引脚时，可以把整个芯片从低功耗状态“吵醒”。

- **软波形模拟：** 如果硬件 I2C 接口不够用了，可以用普通 GPIO 通过软件快速翻转电平，模拟出 I2C 或串口的通信波形（Bit-banging）。
